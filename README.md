# Minitel-Int-Commoditees
COMMODITÉES TRADER . MINITEL STYLE . ON YOUR LOCALHOST:5003 . PROULSED BY YFINANCE .

# 1 . CREATE PYTHON VENV 

     python -m venv com

# 2 . ACTIVATE VENV PYTHON

     source com/bin/activate

# 3 . INSTALL LIBRARIES 

     pip install -r requirements.txt 

# 4 . RUN THE SERVER ON LOCALHOST:5003 

     python serv.py
